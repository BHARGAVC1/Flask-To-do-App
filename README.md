# 🚀 AI-Powered Full Stack Task Manager (Antigravity + Flask)

## 📌 Overview

This project is a **full-stack task management application** built using **Google Antigravity (AI-assisted development)** and **Flask**.

It demonstrates how AI can be used to:

* Plan applications
* Generate code
* Debug and improve systems
* Automate testing workflows

## ✨ Features

### 🖥️ Frontend

* Clean UI using HTML, CSS
* Task creation and deletion
* Interactive task list
* Responsive design

### ⚙️ Backend (Flask)

* RESTful API for task management
* Modular structure (routes, models, utils)
* Configurable app setup
* SQLite database integration

### 💾 Database

* SQLite (`tasks.db`)
* Stores task data persistently

### 🧪 Testing (AI Workflow)

* Auto-generated unit tests
* Workflow: `generate-unit-tests`
* Mock-based testing support

### 🤖 Antigravity Integration

* Code generation using AI
* Custom skill: `code-review`
* Workflow automation
* Debugging with AI assistance


## ⚙️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Flask
* **Database:** SQLite
* **AI Tool:** Google Antigravity
* **Testing:** PyTest (via AI workflows)


### Running the App
1. Navigate to the project folder:
  bash
   cd flask_todo_app
  
2. Install the necessary dependencies:
 bash
   pip install -r requirements.txt
  
3. Run the development server using the factory pattern entry point:
 bash
   python run.py

4. Open `http://localhost:5000` in your browser.

## 🧪 Run Tests

bash
   pytest
